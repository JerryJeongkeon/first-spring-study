## 🙆 실습 코드와 개념 정리

### 게시판 기능

- 게시글 조회
- 게시글 등록
- 게시글 수정
- 게시글 삭제



```
API를 만들기 위해 3개의 클래스가 필요합니다.

1. Request 데이터를 받을 Dto
2. API 요청을 받을 Controller
3. 트랜잭션, 도메인 기능 간의 순서를 보장하는 Service
```
